**YouTube Auto-Skip** adalah aplikasi Android ringan yang dibuat dengan Kotlin dan Jetpack Compose untuk membantu melewati iklan YouTube secara otomatis menggunakan *Accessibility Service*. Cocok untuk kamu yang suka mendengarkan lagu saat bekerja dengan tangan yang kotor/sibuk.

---

## Fitur Utama
* **Auto Skip:** Otomatis menekan tombol "Skip Ad" atau "Lewati Iklan" begitu muncul. Cocok bagi user yang suka putar musik sambil kerja.

## Cara Instalasi
1.  Download APK versi terbaru melalui tombol **Download Latest APK** di atas.
2.  Install APK di smartphone Android kamu.
3.  Berikan izin **Accessibility Service** untuk aplikasi ini di pengaturan HP.
4.  Pastikan izin **Notifications** aktif agar service tetap berjalan di background.
5.  Buka YouTube dan nikmati pengalaman tanpa gangguan klik manual!
